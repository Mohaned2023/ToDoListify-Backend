## ToDoListify Backend:
Simple to-do backend written in **Rust** with **Axum** and **Tokio**.

## Features:
- Login and Register with Sessions.
- Update and Delete the Account.
- Create, Update and Delete Tasks.

## Tech Stack:
- Rust with Axum and Tokio.
- PostgreSQL.
